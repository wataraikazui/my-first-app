# 荻野ラボ プロジェクトガイド

Claude Code向けのプロジェクト概要です。新しいセッションでもここを読めば方針が把握できます。

## プロジェクト概要

- **サイト名**: 荻野ラボ
- **コンセプト**: 家づくりの実験と実践
- **運営**: 荻野 尚志（萩森建設）
- **公開URL**: https://my-first-app-self-eight.vercel.app/
- **ホスティング**: Vercel（GitHub連携で自動デプロイ）
- **技術**: HTML/CSS/JS の静的ファイル（フレームワーク不使用）

## サイト構造

```
index.html                    ← 荻野ラボ トップ（2つの開発室への入口）
├── apps/                     ← アプリ開発室（占い館）
│   ├── index.html            ← アプリ一覧（27アプリ）
│   └── dishwasher-quiz.html  ← 食洗機占い【公開中】
└── methods/                  ← 工法開発室
    ├── index.html            ← 記事一覧（5記事）
    └── （各記事HTML）
```

## 公開済み/準備中の方針

- 全体像（全アプリ・全記事）を最初から一覧表示する
- 公開済みはクリック可能、準備中はグレーアウト＋`pointer-events: none`
- 「準備中」タグで視覚的に区別する

## 各アプリ共通ルール

1. **1アプリ1イシュー** — 機能を詰め込みすぎない
2. **著作権表示** — 全ソースに `© 荻野 尚志 / says@o-h.co.jp` を記載
3. **埋め込み案内** — フッターに「自社サイトに埋め込みたい工務店・ハウスメーカーはご相談ください」
4. **トップへ戻るリンク** — 各アプリのフッターに「← 荻野ラボ トップへ戻る」
5. **Instagramフォローリンク** — 結果画面に「新しい占いの公開はInstagramでお知らせしています」

## 開発フロー

1. 作業ブランチを切る（例: `claude/<task-name>`）
2. 変更をコミット
3. `git push -u origin <branch>`
4. PR を作成して main にマージ
5. Vercel が自動デプロイ

## 参考情報

- **詳細な開発ロードマップ**: 全27アプリ＋5記事のリストは `apps/index.html` と `methods/index.html` を参照
- **食洗機占いのロジック**: `apps/dishwasher-quiz.html` の `gk()` 関数がキー生成、`P`/`C1`/`C5` が結果データ
