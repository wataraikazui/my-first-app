## プロジェクト概要

- **サイト名**: YKTラボ
- **コンセプト**: 家づくりの実験と実践
- **運営**: 荻野 尚志（個人研究所）
- **公開URL**: https://my-first-app-self-eight.vercel.app/

## YKTラボとは

家づくりの実験と実践の場。営業サイトではなく、思想と実績の置き場として運営する個人研究所。
占い・クイズで自分の傾向に気づくツール、実務で培った工法の知見を公開していく。

このサイトのリンクや表記に **萩森建設の名前は出さない**（実践例として別途扱う方針、未確定）。

## 占い・クイズアプリ（占い館シリーズ）共通ルール

YKTラボが開発・公開している占い・クイズアプリは「論理と科学の家づくり占い館」シリーズとして、
萩森建設の本サイト側で実践運用されている5本と同じシリーズの位置づけ。
そのため、アプリ本体のフッター・著作権は占い館シリーズに合わせる。

1. **1アプリ1イシュー** — 機能を詰め込みすぎない
2. **著作権表示** — HTMLコメントに `© 萩森建設 / 論理と科学の家づくり占い館` を記載
3. **フッター** — `有限会社 萩森建設｜このページは、○○のための簡易チェックです。`
4. **占い館リンク** — 各アプリのフッターに「← 占い館にもどる」（本サイトLP: `/lp/fortunelp/`）
5. **利用許諾** — HTMLコメントに `利用は制限しませんが、使用される際は says@o-h.co.jp までご連絡ください`

## 開発フロー

1. 作業ブランチを切る（例: `claude/<task-name>`）
2. 変更をコミット
3. `git push -u origin <branch>`
4. PR を作成して main にマージ
5. Vercel が自動デプロイ

## 参考情報

- **詳細な開発ロードマップ**: 全27アプリ＋5記事のリストは `apps/index.html` と `methods/index.html` を参照
